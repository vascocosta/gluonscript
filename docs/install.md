# From source

1. Fetch the source code from GitHub by cloning the repo with:

```
git clone https://github.com/vascocosta/gluonscript.git
```

2. Compile the source code (you need the `Rust toolchain`):

```
cd gluonscript
cargo build -r
```

3. Copy the generated binary into a location in your PATH

---

# From binaries

1. Fetch the [latest release](https://github.com/vascocosta/gluonscript/releases) from GitHub.

2. Uncompress the archive into a location in your PATH.
